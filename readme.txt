

| Variable Reference |
#--------------------#

id    : 
	a unique identifying number assigned to each locust observed in a single video clip. 

frame : 
	the index of the video frame being analyzed 
	  Unit: video frames (captured at 25 frames per second)

x, y  : 
	coordinates describing the locust's position in space 
	  Unit: cm

state :
	the "motion state" that best describes the locust's motion in each frame.

	  --- Levels ---
	  0: Stationary
	  1: Walking
	  2: Hopping
